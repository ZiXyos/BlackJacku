# Black Jacku
A classic BlackJack game online with classic European rules.
## Getting Started
Clone this repository locally :
```
https://github.com/ZiXyos/BlackJacku.git
```
Install dependencies with npm :
```
npm install
```
## Launch the game
To launch the game, you've to write
```
npm run server
```
